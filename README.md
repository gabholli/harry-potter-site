# Project Title
Harry Potter Universe

## Description
This project contains files for a website about Harry Potter using the following:

- React
- React Router 7
- Tailwind CSS

## Getting Started
### Dependencies
- Runs within one's web browser of choice
### Installing
- This web page does not require any direct installation on your computer
### Executing program
- To execute, navigate within your web browser to https://gabholli-harry-potter-site.netlify.app
